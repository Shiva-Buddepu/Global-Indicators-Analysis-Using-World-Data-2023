# Global Indicators Analysis Using World Data 2023

##  Project Overview
This project presents a comprehensive analysis of global indicators using the **World Data 2023** dataset.  
The study focuses on multiple dimensions—including economy, demographics, environment, health, education, labor, taxation, and urbanization—to understand global development patterns across different countries.

##  Dataset
**File Used:** `world-data-2023.csv`  
The dataset contains country-level information across various socio-economic and environmental domains.
